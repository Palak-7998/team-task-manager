# 🚀 Team Task Manager - MERN Stack

A professional task management system built as part of my final-year technical assessment.

## 🛠 Features
- **Auth Flow:** Secure registration and login.
- **Roles:** Support for `Admin` and `Member` roles.
- **Dashboard:** Real-time task tracking with count updates.
- **MERN Flow:** Full integration between React, Express, and MongoDB.

## ⚙️ Tech Stack
- **React.js** (Frontend UI)
- **Node.js & Express** (Backend API)
- **MongoDB** (Database)
