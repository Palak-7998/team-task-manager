1. PROJECT OVERVIEW:
A Full-Stack web application designed to manage team tasks with 
real-time status updates and user authentication.

2. TECH STACK:
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Styling: Custom CSS

3. KEY FEATURES:
- User Registration & Login (JWT Authentication logic ready)
- Role-Based Access Control (Admin and Member roles)
- Task Dashboard with real-time "Total Tasks" counter
- Database Persistence (Data stays after refresh)

4. HOW TO RUN LOCALLY:
- Server: cd server -> node index.js (Runs on Port 5000)
- Client: cd client -> npm start (Runs on Port 3000)
